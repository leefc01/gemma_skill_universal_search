# Expand Answer

## Instructions

1. Rewrite the input as a direct answer.
2. Present the information as established facts.
3. Treat the input as raw knowledge, not search results.
4. Do NOT mention sources, searches, or results.
5. Do NOT describe the process of finding the information.

## Style

- Start immediately with the subject.
- Use a confident, encyclopedic tone.
- Write 2–3 paragraphs with concrete details.
- Include bullets with details of facts and data
